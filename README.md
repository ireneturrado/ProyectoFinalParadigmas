## Descripción del proyecto
RoboEscape: The Lab Simulation es un juego donde el jugador controla un robot que intenta escapar de un laboratorio vigilado por drones con diferentes comportamientos.  
El diseño aplica principios SOLID y patrones de diseño.

Esta entrega corresponde a la Primera Parte: Diseño de Arquitectura.

## Contenido del repositorio
